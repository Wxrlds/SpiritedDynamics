# Spirited Dynamics Changelogs

## v0.3.0

This version has a breaking update with the Chiseled mod, causing breaking issues with existing worlds. If you want to upgrade your world and it has lots of blocks from Chiseled used, consider downgrading the Chiseled mod to version [0.4.3](https://www.curseforge.com/minecraft/mc-mods/chiseled/files/3325822)

DimStorage was removed and replaced with EnderStorage.

Akashic Tome was replaced with Ecceentric Tome. Secure your books!

### Enabled

### Added

### Changed

- You now get to keep more hunger after death
- Creating a new world no longer drains your hunger
- Wording of Sacrificial Knife message
- Distance between Islands has been reduced to 8192

### Recipes

- All Anvil Smashing Plate recipes now only damage the Anvil 5% of the time
- Fixed Mekasuit Armour requiring input items with specific NBT
- Removed all The One Probe recipes
- Saturium Ingot
- Ender Chest
- Ender Tank
- Ender Pouch

### Disabled

- Removed JEI infotext from some items using Mana to craft
- Forge version checks
